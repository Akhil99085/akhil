# 🎂 Birthday Surprise Webpage 🎂

A beautiful, interactive birthday surprise webpage with professional animations and effects!

## ✨ Features

### 🔐 Password Protection
- Enter date of birth (28/08/2006) to unlock the surprise
- Beautiful error messages with shake animation

### 💓 Heart Animation
- Stunning blinking heart with glow effects
- Stars in the background
- Click to proceed to the next step

### 🎆 Loading Screen
- Impressive fireworks animation
- Professional loading spinner
- Smooth transitions

### 🎉 Birthday Wishes
- Celebratory birthday message
- Confetti cannon animation with 100+ pieces
- Floating balloons
- "Open My Heart" button for next step

### 💌 Birthday Card
- Beautiful 3D card flip animation
- Front: Birthday greeting
- Back: Heartfelt special message
- Click to flip between front and back

## 🚀 How to Use

1. Open `index.html` in your browser
2. Enter the password: `28/08/2006`
3. Click the blinking heart
4. Watch the fireworks and confetti!
5. Click "Open My Heart" button
6. Click the birthday card to flip it

## 📁 Files

- `index.html` - Main webpage structure
- `styles.css` - All styling and animations
- `script.js` - Interactive functionality

## 💝 Customization

You can easily customize:
- **Password**: Change `correctPassword` in script.js
- **Birthday Message**: Edit the wishes message in index.html
- **Special Lines**: Modify the card message text
- **Colors**: Update gradient colors in styles.css
- **Name**: Add recipient's name to any section

## 🎯 Browser Support

Works on all modern browsers:
- Chrome/Edge
- Firefox
- Safari
- Mobile browsers

## 📝 Notes

- Make sure all three files (HTML, CSS, JS) are in the same directory
- No external libraries required - pure HTML/CSS/JavaScript
- Audio is generated programmatically (no audio files needed)
